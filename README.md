# Payroll-App-With-React

This is a payroll application which allows the user to calculate the net salary (without taxes) of an employee per year. Additionally, it also provides functionality for CRUD operations which includes registering a new employee, updating an existing employee information, or removing an existing employee. Front-end of the application is designed using ReactJs and the Back end is implemented in C# using .NET Framework Web API.


Prerequisites:

For this payroll application to run properly, please execute “npm install” command so that it creates a node_modules folder in your project with all the required dependencies. 

For the API to work properly, please restore the packages defined in the packages.config file by using NuGet Package Manager.
